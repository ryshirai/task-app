# GlanceFlow

チームの「今」を一瞥で把握する。視認性特化型チーム・タイムライン・ダッシュボード。

## 1. 概要

**GlanceFlow** は、チームメンバーの稼働状況を直感的なタイムライン形式で可視化し、コミュニケーションコストを削減するためのツールです。「誰が」「今」「何をしているか」を把握するために、もはや誰かに尋ねる必要はありません。

## 2. 主な機能

### 📅 インタラクティブ・タイムライン
- **9:00〜18:00の稼働表示**: 業務時間に特化した高密度なガントチャート。
- **ドラッグ＆ドロップ操作**: マウス操作でタスクの作成、時間の移動、期間の変更が完結します。
- **日付切り替え**: 過去の履歴や未来の予定もカレンダー形式でスムーズに確認。

### 🏷️ タスク管理 & タグ
- **進捗の可視化**: 実行中タスクの進捗率をプログレスバーで表示。
- **タグ機能**: プロジェクト名や「会議」「緊急」などのタグを付与し、瞬時にフィルタリング可能。

### 📊 チーム負荷の可視化（ワークロード）
- **負荷レベル判定**: タスクの重複や件数から、「集中」「多忙」「余裕」といったステータスを自動表示。
- **オーバーワーク防止**: 重複したスケジュールを赤色で強調し、無理な計画を即座に発見。

### 📝 日報システム
- **自動テンプレート**: その日のタスク内容から、Markdown形式の日報を下書きとして自動生成。
- **共有 & 編集**: チーム全員の日報を一覧表示。自分の投稿は後から修正も可能です。

### 🔒 エンタープライズ・セキュリティ
- **マルチテナント対応**: 組織ごとに独立したワークスペースを構築。
- **招待制の導入**: 管理者が発行する招待URLを通じてのみ、新しいメンバーが参加可能。
- **操作履歴 (Activity Log)**: 「いつ」「誰が」「何を」変更したかをすべて記録。

## 3. 技術スタック

- **Frontend**: SvelteKit, Tailwind CSS, TypeScript
- **Backend**: Rust (Axum), SQLx (PostgreSQL), Argon2, JWT
- **Database**: PostgreSQL
- **Environment**: Docker / Docker Compose

## 4. クイックスタート

### 4.1. バックエンドの起動
```bash
cd backend
# .env ファイルに DATABASE_URL, JWT_SECRET を設定
cargo run
```

### 4.2. フロントエンドの起動
```bash
cd frontend
npm install
npm run dev
```

## 5. 基本的な使い方フロー

1.  **組織の作成**: ログイン画面の「新しい組織を作成する」から、あなたのチームのワークスペースを作成します。
2.  **メンバーの招待**: 右上の「ユーザー管理」アイコンから招待URLを発行し、チームメンバーに共有します。
3.  **タスクの登録**: 自分の行の空いている場所をドラッグして、今日の予定を登録します。
4.  **状況の共有**: 画面を見ているだけで、他のメンバーがタスクを更新した状況が（ポーリングにより）数秒以内に反映されます。
5.  **一日の終わりに**: 「日報提出」ボタンを押し、自動生成された内容を確認して提出します。
